# Project Title: Weapon Detection System (Project-25)
This project is a Machine Learning–based Weapon Detection System designed to identify weapons such as firearms in images and video streams. The system uses computer vision and deep learning techniques to analyze visual data and detect potential threats in real time. It aims to support security and surveillance applications by improving monitoring efficiency and response time.
Features

Detects weapons in images and video feeds

Supports real-time camera input

Uses machine learning and computer vision models

Web-based interface for visualization

Modular backend and frontend structure

Scalable and easy to extend
Tech Stack

Programming Language: Python, JavaScript

Frontend: HTML, CSS, JavaScript

Backend: Python (Flask/FastAPI/Custom APIs)

Machine Learning: OpenCV, Deep Learning Models

Other Tools: Git, REST APIs
